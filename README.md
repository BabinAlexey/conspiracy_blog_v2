# ConspiBlog — блог о конспирологических теориях

## Описание

Современный одностраничный сайт-блог без серверной части и БД. Все данные (комментарии) хранятся в браузере пользователя (localStorage).

## Страницы

- **Главная** — список теорий
- **О блоге** — информация о проекте
- **Теории** — подробные страницы трёх теорий
- **Контакты** — форма обратной связи

## Формы

- **Обратная связь** (на странице "Контакты")
- **Комментарии** (на страницах теорий)

## Установка и запуск

1. **Скачайте архив или клонируйте репозиторий**
2. **Откройте файл `index.html` в браузере**  
   (двойной клик или через контекстное меню "Открыть с помощью...")

> Сайт работает полностью локально, интернет не требуется.

## Структура файлов

- `index.html` — главная страница
- `about.html` — о блоге
- `theory1.html`, `theory2.html`, `theory3.html` — страницы теорий
- `contact.html` — контакты
- `style.css` — стили
- `main.js` — логика (комментарии, формы)
